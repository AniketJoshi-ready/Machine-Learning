# Machine Learning Projects

This repository contains a collection of **Machine Learning projects and case studies** implemented in Python. Each project demonstrates the end-to-end workflow of data preprocessing, model training, evaluation, and visualization. The aim is to provide practical, real-world examples of applying machine learning algorithms to diverse domains.

---

## 📌 Projects

### 1. House Price Prediction
Regression-based model for predicting real estate property prices.

### 2. Iris Classifier – Decision Tree & KNN
Classifies Iris flower species using classic ML models.

### 3. Breast Cancer Detection – Logistic Regression 
Binary classification of breast cancer tumors for early detection.

---

## 🚀 Features

* Covers **classification, regression, clustering, and ensemble learning**.
* Implements popular ML algorithms: Decision Trees, Logistic Regression,KNN, Random Forest, etc.
* Practical case studies with datasets
